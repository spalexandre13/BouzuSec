# test1

Site web hébergé sur **GitHub Pages** :  
🔗 **https://spalexandre13.github.io/test1/**

## Structure

| Fichier | Rôle |
|---------|------|
| `index.html` | Page principale du site |
| `style.css` | Feuille de styles |
| `.github/workflows/deploy.yml` | Déploiement automatique via GitHub Actions |

## Activer GitHub Pages (une seule fois)

1. Aller dans **Settings → Pages** du dépôt.
2. Dans *Source*, choisir **GitHub Actions**.
3. Sauvegarder — le site sera en ligne après le prochain push sur `main`.

> Chaque push sur la branche `main` déclenche automatiquement le workflow et met à jour le site.
